# Agent Instructions

## Media Page (`media.html`)

Whenever design assets are added, removed, or renamed in the `design/` directory (logos, business cards, or any future assets), the `/media` page (`media.html`) **must be updated** to reflect the changes.

This includes:
- Adding new asset cards when new files are added
- Removing entries when files are deleted
- Updating names, descriptions, or download links when files are renamed
- Adding new sections if new asset categories are introduced under `design/`

The media page serves as a marketing-accessible index of all brand and design assets.
